Quick Rules

Navy, Not Black
Background is #050c1a — deep navy. Never bg-black or pure #000000.
Cyan Borders
Default border = rgba(0,212,255,0.13). Active/hover = rgba(0,212,255,0.38). Not solid lines.
Tracker = Accent
Tracker type colors are UI accents. Purple → agents/AI. Cyan → primary actions. Gold → mood. Neon green → nutrition.
Numbers → Mono
All scores, values, data: font-mono (Share Tech Mono). Gradient fill on hero numbers: cyan → purple.
Labels → Audiowide
Every field label, button, tab, badge uses font-ui. Uppercase, 9–11px, letter-spacing 0.10–0.18em.
Titles → Chakra Petch
font-display-heading auto applies bold + italic + uppercase. Lean into it — never override the transform.
Glow Effects
Active swatches: drop-shadow(0 0 16px color 45%). Active borders: add box-shadow: var(--glow).
Font Roadmap
Current free fonts swap to premium when purchased: Chakra Petch → Cyberhype, Share Tech Mono → Unhuman, Audiowide → Omnitrinx.
