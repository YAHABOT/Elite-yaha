Quick Rules

OLED First
Never use bg-white or bg-gray-*. Always use surface tokens.
Tracker = Accent
Tracker type colors double as UI accents. Purple = agents/AI. Cyan = primary actions.
Numbers → Mono
All scores, values, and data points use font-mono.
Uppercase Titles
font-display-heading auto-uppercases. Don't fight it — lean in.
Borders
Default border-border. Hover states use white/10. Colored borders use color/20–30.
Glow Effects
Use drop-shadow or box-shadow with tracker color at 20–40% opacity.
